Chartbeat Publishing Setup

Getting up and running with Chartbeat Publishing is a simple process but does require some familiarity with your site’s architecture and JavaScript. The instructions on this page will guide you through the implementation process for a new domain. For reference, the full Chartbeat Publishing documentation is available in the Docs section of the support site.

First select the additional product(s) that you are implementing on your site.



Generate Javascript


Chartbeat Publishing Code — Implement on all pages on your site.

Part 1

Part 1: Insert this right after the opening <head> tag:

<script type='text/javascript'>var _sf_startpt=(new Date()).getTime()</script>
Part 2

Insert this script right before the closing <body> tag:

<script type='text/javascript'>
var _sf_async_config = _sf_async_config || {};
/** CONFIGURATION START **/
_sf_async_config.uid = 12345
_sf_async_config.domain = 'isthatoffensive.tumblr.com';
_sf_async_config.useCanonical = true;
_sf_async_config.videoType = 'ooyala';
_sf_async_config.sections = 'Change this to your Section name'; //CHANGE THIS
_sf_async_config.authors = 'Change this to your Author name'; //CHANGE THIS
/** CONFIGURATION END **/
(function() {
function loadChartbeat() {
window._sf_endpt = (new Date()).getTime();
var e = document.createElement('script');
e.setAttribute('language', 'javascript');
e.setAttribute('type', 'text/javascript');
e.setAttribute('src', '//static.chartbeat.com/js/chartbeat_video.js');
document.body.appendChild(e);
}
var oldonload = window.onload;
window.onload = (typeof window.onload != 'function') ?
loadChartbeat : function() {
oldonload();
loadChartbeat();
};
})();
</script>

For more information about setting the section and author variable on lines 6 and 7 see the section and author docs on the Support Site.
