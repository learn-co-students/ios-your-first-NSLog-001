Chartbeat Publishing Setup

Getting up and running with Chartbeat Publishing is a simple process but does require some familiarity with your site’s architecture and JavaScript. The instructions on this page will guide you through the implementation process for a new domain. For reference, the full Chartbeat Publishing documentation is available in the Docs section of the support site.

A few definitions of Chartbeat implementations to get you started.

Synchronous

Does not utilize the window.onload function and will not load asynchronously — meaning that it will load inline with all other elements on the page.

Asynchronous

Does not utilize the window.onload function and will not load asynchronously — meaning that it will load inline with all other elements on the page.

document.ready

Does not utilize the window.onload function and will not load asynchronously — meaning that it will load inline with all other elements on the page.

                (function() {
                    function loadChartbeat() {
                        window._sf_endpt = (new Date()).getTime();
                        var e = document.createElement('script');
                        e.setAttribute('language', 'javascript');
                        e.setAttribute('type', 'text/javascript');
                        e.setAttribute('src', '//static.chartbeat.com/js/chartbeat.js');
                        e.setAttribute('src', '//static.chartbeat.com/js/chartbeat_video.js');
                        document.body.appendChild(e);
                    }
                    var oldonload = window.onload;
                    window.onload = (typeof window.onload != 'function') ?
                        loadChartbeat : function() {
                            oldonload();
                            loadChartbeat();
                        };
                })();
            </script>
